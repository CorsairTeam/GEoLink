1.Onglet Points

1.1 Création d'un point "Coordonnées"

Le menu déroulant permet de choisir le format des coordonnées pour la création d'un point :

  • Coordonnées en degrés décimaux.
  • Coordonnées en degrés, minutes.
  • Coordonnées en degrés, minutes et secondes.
  • Coordonnées Calamar.

Les différentes possibilitées sont représentées dans la figure ci dessous :

Les champs de saisies permettent de paramètrer les propriétés du point :

  • Nom : Nom du point.
  • Taille :

    Taille du point, ce paramètre est pris en compte dans les exports kml vers Foreflight.

    Une valeur de 1 correspond à la valeur par défaut dans ForeFlight.

    Une valeur de 0,7 soit 70% de la taille par défaut est une valeur recommandée pour l'affichage sur tablette.

  • Couleur : Couleur du label lié au point, uniquement pris en compte dans Foreflight.
  • Type de point : Menu déroulant permettant de choisir la forme du point qui sera affiché dans Foreflight.
  • Latitude : Latitude du point.
  • Longitude : Longitude du point.
  • Axe X : Permet de définir les coordonnées du point en mètres avant (mC) ou aprés (mL) la croix centrale (Axe 236°).
  • Axe Y : Permet de définir les coordonnées du point en mètres à droite (mD) ou à gauche (mG) de l'axe de Calamar (Axe 236°).
  • Créer le point : Crée le point avec les paramètres sélectionnés.

SDVFR ne tient pas compte des paramètres de personnalisation. Il est nécessaire de configurer les tailles, couleurs des lignes, points et polygones aprés l'export sur la tablette.

1.2 Création d'un point "Radial Distance"

Ce mode permet de créer un point situé dans un Radial / Distance donné d'un point de référence.

  • Point de départ :

    Liste déroulante permettant de choisir le point de référence dans la liste des points actuellement créés

  • Distance :

    Permet de choisir la distance par rapport au point de référence en mètres ou en nautiques.

  • Gisement :

    Permet de choisir le gisement (en degrés) du nouveau point par rapport au point de référence.

1.3 Création d'un point : "Création sur la carte"

  • Clic gauche maintenu + déplacement de la souris : Déplacement de la carte.
  • Molette : Zoom avant / zoom arrière.
  • SHIFT + Clic gauche sur la carte permet de sélectionner un point (en bleu sur la carte). Les champs latitude et longitude sont alors automatiquement renseignés. L'appui sur le bouton "Créer le point" génére le point.

1.4 Liste des points

Ce tableau liste l'ensemble des points créés :

  • La case à cocher permet de sélectionner le point pour effacement ou export.
  • Select All Sélectionne tous les points.
  • Deselect All Désélectionne tous les points.
  • Delete Efface les points sélectionnés.

Click Gauche sur le nom du point sélectionne les coordonnées en degrés et met à jour les champs avec les paramètres du point. On peut alors modifier les paramètres du point et les enregistrer en appuyant sur le bouton Créer le point.

Double-click Gauche sur le nom du point centre la carte sur le point.

2. Onglet Lignes

2.1 Création d'une ligne : "Création par sélection de points"

Permet de créer une ligne en sélectionnant des points dans la liste des points créés.

  • Sélectionner un point dans la boîte Points et l'ajouter à la ligne via le bouton >>.
  • On peut retirer un point de la ligne en le sélectionnant et en cliquant sur le bouton <<.
  • Pour réordonner les points dans la ligne, on effectue un Click Gauche maintenu sur le point a déplacer et on le replace dans la ligne.
  • Nom : Nom de la ligne.
  • Épaisseur de la ligne : Choix de l'épaisseur de la ligne (6 conseillé pour ForeFlight).
  • Couleur : Choix de la couleur de la ligne.
  • L'appui sur le bouton Créer la ligne crée la ligne avec les paramètres sélectionnés.

2.2 Création d'une ligne : "Création sur la carte"

Cett option permet de créer la ligne en la dessinant directement sur la carte.

  • SHIFT + CLIC GAUCHE Crée un nouveau point sur la carte.
  • SHIFT + CLIC DROIT Efface le dernier point ajouté.
  • CLIC GAUCHE maintenu sur la ligne permet de translater la ligne sur la carte.
  • SHIFT + CLIC GAUCHE sur un point existant permet de le sélectionner et de le déplacer.

Le champ Longueur totale affiche la longueur totale de la ligne en mètres ou nautiques.

Le champ Longueur du dernier segment affiche la longueur du dernier segment ajouté en mètres ou nautiques.

Le champ Gisement du dernier segment affiche le gisement du dernier segment ajouté en degrés.

2.3 Liste des lignes

  • Select All Sélectionne toutes les lignes.
  • Deselect All Désélectionne toutes les lignes.
  • Delete Efface les lignes sélectionnées.
  • Double-clic sur le nom de la ligne centre la carte sur la ligne.
  • CLIC GAUCHE sur la case à cocher sélectionne la ligne pour effacement ou export.

3. Onglet Polygones

3.1 Création d'un polygone : "Utilisation des points de la base de données"

Permet de créer un polygone en sélectionnant des points dans la liste des points existants.

  • Ajouter un point en le selectionnant et appuyer sur le bouton >> pour le transférer vers le polygone.
  • Retirer un point du polygone en le sélectionnant et en cliquant sur le bouton <<.
  • Pour réordonner un point dans le polygone, effectuer un Click Gauche Maintenu sur ce point et le repositionner dans le polygone.

Les champs de saisie permettent de définir les propriétés du polygone :

  • Nom : Nom du polygone.
  • Épaisseur de la ligne : Epaisseur de la ligne (6 conseillé pour ForeFlight).
  • Couleur du contour : Couleur de la ligne externe du polygone.
  • Couleur du remplissage : Couleur du remplissage du polygone. Nécessite de cocher la case "Remplir" pour prise en compte de la couleur de remplissage.
  • Créer le polygone : Création du polygone avec les paramètres sélectionnés.

3.2 Création du polygone : "Tracé du polygone sur la carte"

Permet de créer le polygone en le dessinant directement sur la carte.

  • SHIFT + CLIC GAUCHE permet de créer un nouveau point sur la carte.
  • SHIFT + CLIC DROIT efface le dernier point ajouté.
  • CLIC GAUCHE (Maintenu) sur la ligne permet de la translater sur la carte.
  • SHIFT + CLIC GAUCHE (Maintenu) sur un point existant permet de le sélectionner et de le déplacer.

Le dernier point peut se positionner par rapport au point précédent en s'aidant des champs :

  • Longueur du dernier segment : longueur du dernier segment ajouté en mètres ou nautiques.
  • Gisement du dernier segment : gisement du dernier segment ajouté en degrés.

3.3 Création d'un polygone : "Création d'une boite"

  • Centre de la boite :Choisir dans la liste des points l'origine de la boite.
    • ²
  • Orientation :Choisir l'orientation de la boite en degrés.
  • Longueur vers l'avant :Extension de la boite dans le sens de l'orientation en mètres ou nautiques.
  • Longueur vers l'arrière :Extension de la boite dans le sens opposé à l'orientation.
  • Longueur vers la gauche :Extension de la boite à gauche perpendiculairement à l'orientation.
  • Longueur vers la droite :Extension de la boite à droite perpendiculairement à l'orientation.

La case à cocher Ajouter une flèche permet d'ajouter une flèche indiquant graphiquement la direction de la boite (Stationnaires).

3.4 Création d'un cercle

Permet de créer un cercle ou un arc de cercle :

  • Choisir dans la liste des points le centre du cercle.
  • Sélectionner un rayon en mètres ou nautiques.
  • Sélectionner le nombre de segments pour tracer le polygone.

La case à cocher Arc de cercle ouvre la boite de saisies pour generer un arc de cercle :

  • Angle de départ :Entrer l'angle de départ entre 0 et 360°.
  • Angle d'arrivée :Entrer l'angle d'arrivée entre 0 et 360°.
  • L'arc part de l'angle de départ et rejoint l'angle d'arrivée dans le sens horaire.
  • La case à cocher Relier au centre permet de relier l'arc au centre (Arc jaune).
  • Si la case n'est pas cochée, une branche relie les deux extrémités de l'arc (Arc vert)

Gestion des polygones

  • Select All permet de sélectionner tous les polygones.
  • Deselect All permet de désélectionner tous les polygones.
  • Delete permet d'effacer les polygones sélectionnés.
  • Double-clic sur le nom du polygone centre la carte sur le polygone.
  • CLIC GAUCHE sur la case à cocher sélectionne le polygone pour effacement ou export.

Import / Export

Importer fichier KML

Permet d'importer un fichier KML.

Importer fichier CSV

Permet d'importer un fichier CSV possédant les champs Name et Lat,Lon exprimés en degrés décimaux.

Exemple de structure de fichier CSV pour l'import :
Name Lat Lon
Point A 44.123456 -1.234567
Point B 43.987654 -1.543210

Exporter vers SDVFR

Permet d'exporter les objets sélectionnés vers SDVFR, pas de prise en compte des champs taille, couleur et type de point.

Exporter vers ForeFlight

Permet d'exporter les objets sélectionnés vers ForeFlight avec prise en compte des champs taille, couleur et type de point.

Les fichiers KML permettent d'exporter des points, des lignes et des polygones.

Les fichiers CSV permettent d'exporter uniquement des points.

Export ForeFlight

Pour un export ForeFlight, utiliser le nom de fichier user_waypoints.csv.

Les exports ForeFlight prennent en compte les champs taille, couleur et type de point lorsque ces informations sont disponibles.

Structure d'export recommandée :

WAYPOINT_NAME, Waypoint description, Latitude, Longitude, Elevation

Exemple de ligne CSV ForeFlight :
WAYPOINT_NAME Waypoint description Latitude Longitude Elevation
POINT_A Exemple de waypoint 44.123456 -1.234567 0
